# CityLibraryFund

Информационная система для автоматизации работы библиотечного фонда города.

## О проекте

CityLibraryFund — desktop-приложение, разработанное на платформе WPF с использованием Entity Framework Core и SQL Server.
Система предназначена для автоматизации процессов учета библиотечного фонда, работы с читателями, выдачи литературы, посещений библиотек и аналитики.

Приложение поддерживает разграничение прав доступа и работу нескольких ролей пользователей.

---

# Основные возможности

## Администратор

* управление библиотеками и читальными залами;
* управление сотрудниками;
* управление авторами и произведениями;
* управление изданиями и экземплярами фонда;
* учет поступлений и списаний литературы.

## Библиотекарь

* регистрация читателей;
* оформление выдачи и возврата литературы;
* учет посещений;
* просмотр истории выдач;
* работа с экземплярами фонда.

## Аналитик

* просмотр статистики библиотеки;
* анализ популярных произведений;
* анализ просроченной литературы;
* аналитические запросы по читателям, фонду и выдачам;
* статистика работы сотрудников.

---

# Архитектура приложения

Проект построен по многослойной архитектуре:

* CityLibraryFund.Data — слой данных;
* CityLibraryFund.WPF — пользовательский интерфейс;
* Services — бизнес-логика;
* DTO — объекты передачи данных;
* Entity Framework Core — работа с базой данных.
